# SampleMosbyMVI
Simple example of clean architecture Android app with MVI using Mosby (with State Reducer).

## Requirements
* Android Studio 3.5 Canary 6
* Kotlin 1.3.21
* Android Gradle Plugin 3.5.0-alpha06
* Gradle wrapper 5.2.1
* AAPT 2
